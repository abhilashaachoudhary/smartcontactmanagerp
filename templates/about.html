About Smart Contact Manager

A secure and user-friendly contact management system built with Flask, offering seamless authentication, email verification, and role-based access for both users and administrators.

Key Features

  • Dynamic registration with email verification
  • Role-based dashboards for Admin and Users
  • Google and GitHub OAuth login support
  • Secure password hashing and session management
  • Admin access to create, update, and delete contacts

Why This Project?

This project is inspired by Spring Boot Security principles and built using Python Flask to demonstrate a robust full-stack web application. It showcases real-world practices like user role segregation, CRUD operations, and third-party authentication integration.

Ideal for learning secure web development practices, this project is scalable and can be extended into a full-fledged CRM or integrated into larger applications.

Meet the Developer

Developed as part of an MTech project to demonstrate secure user access, contact management, and full-stack development using Python, Flask, and OAuth.
